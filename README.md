# IT资产管理系统 python81

## 项目概述

IT资产管理系统是基于Python实现的现代化资产管理平台。它采用Django作为后端框架，Bootstrap作为前端设计，为数据中心、服务器、网络设备、安全设备等提供全生命周期的管理。

## 技术栈

- **开发语言：** Python 3.11.2
- **后端框架：** Django
- **前端框架：** Bootstrap
- **数据库：** MySQL 8.x

## 功能模块

### 基础功能

- **首页：** 提供系统可视化统计。
- **数据中心管理：** 对各类数据进行集中管理。
- **网络设备管理：** 实现网络设备的统一管理。
- **服务器管理：** 对服务器进行全生命周期管理。
- **存储设备管理：** 管理企业存储设备。
- **安全设备管理：** 确保安全设备有效运行。
- **用户管理：** 系统用户及其权限管理。

### 高级功能

- **批量导入/导出：** 支持Excel格式，方便数据处理。
- **批量操作：** 支持批量删除，提高操作效率。

## 系统角色

- 系统管理员
- 数据中心管理员
- 网络设备管理员
- 服务器管理员
- 存储设备管理员
- 安全设备管理员
